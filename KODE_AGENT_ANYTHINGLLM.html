KODE AGENT

AnythingLLM Integrasjon

FOR LOKAL AI BRUK!
Koble Kode Agent til din private AnythingLLM/Ollama installasjon. Ingen data sendes til skyen!

🚀 HURTIGSTART (3 steg)

STEG 1: Start API Server

Dobbeltklikk: START_KODE_AGENT_SERVER.bat på skrivebordet

Eller kjør i terminal:

cd C:\Sandkasse_Produksjon python anythingllm_agent.py

Server starter på: http://127.0.0.1:8765

STEG 2: Konfigurer AnythingLLM

1. Åpne AnythingLLM (192.168.1.200:3001)

2. Gå til Settings → LLM Preference

3. Velg Ollama (din lokale modell)

4. Lim inn System Prompt (fra anythingllm_system_prompt.txt)

STEG 3: Test!

I AnythingLLM chatten, skriv:

Opprett et Python-prosjekt kalt "MinTestApp"

AI-en skal:

  • Tolke kommandoen
  • Kalle API-et
  • Opprette prosjektet
  • Gi deg filstien

🔗 API ENDPOINTS

POST http://127.0.0.1:8765/command

Hovedkommando for alle handlinger

{ "action": "create", "project_name": "MinApp", "description": "En test app", "project_type": "python" }
GET http://127.0.0.1:8765/

Health check - sjekk om server kjører

GET http://127.0.0.1:8765/projects

Liste alle aktive prosjekter
