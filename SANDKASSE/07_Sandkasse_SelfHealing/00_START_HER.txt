Problem: "Kan ikke koble til HTTP-server"
→ Sjekk: http://100.114.112.61:8000/ i nettleser på Lenovo
→ Sjekk: ping 100.114.112.61

Problem: "curl: (7) Failed to connect"
→ Acer må være på og HTTP-server må kjøre
→ Prøv: python -m http.server 8000 (i for_lenovo/ mappen)

Problem: "Permission denied" på install.sh
→ Kjør: chmod +x install.sh

Problem: "python3: command not found"
→ Kjør: sudo apt install python3 python3-pip
